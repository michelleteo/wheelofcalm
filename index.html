OPENING REMARKS

This emotion regulation toolkit is to be used by a school staff (the user) to HELP a student MANAGE his/her INTENSE emotions in school. The user should familiarize himself/herself with the WHOLE toolkit. Background knowledge of the student's distress/anger is helpful but not necessary in using this toolkit. Designers of this toolkit believe that : - the user is the MORE important factor in helping the student. - this toolkit is NOT meant as a SOLUTION to manage uncooperative/defiant behaviour in school. - the user will use it at his/her discretion
