Wheel of Calm

1)OPENING REMARKS

This emotion regulation toolkit is to be used by a school staff (the user) to HELP a student MANAGE his/her INTENSE emotions in school.

The user should familiarize himself/herself with the WHOLE toolkit.

Background knowledge of the student's distress/anger is helpful but not necessary in using this toolkit.

Designers of this toolkit believe that :

- the user is the MORE important factor in helping the student.

- this toolkit is NOT meant as a SOLUTION to manage uncooperative/defiant behaviour in school.

- the user will use it at his/her discretion

2)INFORMATION/ INSTRUCTIONS

A) Introduction

B) Aims and Objectives

This toolkit aims to help the student to regulate his /her emotions towards a calmer state.

SOLER:

Using of SOLER is optional for the user as it focuses on attending skills in a session with student.

S - Sitting at a comfortable angle and distance.

O - Open posture. Arms and legs uncrossed.

L - Leaning forward from time to time. Looking genuinely interested. Active listening.

E - Effective eye contact without staring.

R - Remaining relatively relaxed.
